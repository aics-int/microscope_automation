3i ImportsÂ¶

There is a submodule, imports, which puts all the imports for a single microscope into one statement. For example, by running the below code you will import hardware_control_3i as hardware_control, slidebook_experiment_info as experiment_info, and all the 3i connectors, i.e. connect_slidebook under the name slidebook.

import microscope_automation.imports.intelligent_imaging_innovations as slidebook
slidebook.connect_slidebook.ConnectMicroscope()
